# ComplaintCare (ResolveNow) - Backend

Node.js + Express + MongoDB backend for the Online Complaint Registration and Management System.

## Prerequisites

- Node.js
- MongoDB (local or connection string in env)

## Setup

1. Install dependencies: `npm install`
2. Set `MONGO_URI` in env if not using default `mongodb://localhost:27017/complaintcare`
3. Start server: `npm start` (uses nodemon)

Server runs on `http://localhost:8000`. Frontend proxy should point here.
